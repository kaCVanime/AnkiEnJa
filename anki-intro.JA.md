# 功能

1. 通过*随机例句*学习`“日语 -> 中文”`与`“中文 -> 日语”`
2. 不仅学习一个词的多种释义，也会掺入学习各种*用法*(习语、复合语等)
3. 对于一个单词所已熟练的释义/用法，可点击句子取消高亮，该释义/用法不会在本次学习的后续中出现

# 可配置项

#### 字太小/太大

1. Anki首页，点击“牌组”
2. 点击所要改变字体大小的牌组名称
3. 点击“浏览”
4. 点击右侧的"卡片..."按钮
5. 点击左侧的“样式”按钮
6. 找到`.root { `
7. 把`transform: scale(0.8)`中的`0.8`改成想要的缩放率(例. 放大1.5倍则改成1.5)
8. 点击右下的保存
