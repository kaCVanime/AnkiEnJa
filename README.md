# AnkiEnJa
英语语境智能卡组  
日语语境智能卡组

# 亮点
1. 针对"一个词可能有多种释义、词组、习语"，将其全部拆成独立卡片。**一卡一释义**，最大化记忆效率。
2. 每次学习展示**随机例句**。对填鸭学习说不，配合语境理解单词。
3. 通过AI生成基于**词义**的使用频率，并对其分类。超高可定制化，只学习感兴趣的领域和高低频词。
4. 将"阅读理解"和"写作"拆分成**独立子卡组**。给不同的学习活动分配特有的记忆参数，进一步优化复习效率。
